# Setup

## Requirements

* Python 3.6+

## Installation

Install intercessor with pip:

```sh
$ pip install intercessor
```

or directly from the source code:

```sh
$ git clone https://github.com/RadicalZephyr/intercessor.git
$ cd intercessor
$ python setup.py install
```
